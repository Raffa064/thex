# Editor

| Keys               | Control            |
|--------------------|---------------------|
| Arroys, j, k, i, l | Move cursor         |
| s                  | Toggle Selection    |
| [0-9][a-f]         | Hex Insert (nibble) |

# Command line 

| Key | Description          |
|-----|----------------------|
|  :  | Open command line    |
|  ;  | Clear command output |

| Command   | Description          |
|-----------|----------------------|
| :q        | Quit                 | 
| :m[bnm]   | Place marker         |
| :mr       | Remove marker        |

# Editor

| Keys               | Control          |
|--------------------|------------------|
| Arroys, j, k, i, l | Move cursor      |
| s                  | Toggle Selection |

# Command line 

| Key | Description          |
|-----|----------------------|
|  :  | Open command line    |
|  ;  | Clear command output |

| Command   | Description          |
|-----------|----------------------|
| :q        | Quit                 | 
| :m[bnm]   | Place marker         |
| :mr       | Remove marker        |
